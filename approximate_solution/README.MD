To run the tests cases: 
python3 approximate_solution.py test_cases/small_graph.txt
python3 approximate_solution.py test_cases/medium_graph.txt
python3 approximate_solution.py test_cases/large_graph.txt
python3 approximate_solution.py test_cases/non_optimal.txt

To run the wallclock tests please go to the test-cases directory, open the compute_approx_wallclock directory and then run:

chmod +x compute_approx_wallclock.sh
./compute_approx_wallclock.sh
